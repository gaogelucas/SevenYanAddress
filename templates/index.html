API 使用说明

本应用同时提供了 RESTful API，允许程序化方式交互:

  • 经纬度转七言地址: POST /api/hanzi/
  • 七言地址转经纬度: POST /api/coordinates/

API 使用示例:

curl -X POST -H "Content-Type: application/json" -d '{"latitude": 31.2304, "longitude": 121.4737}' https://sevenyan/api/hanzi/
curl -X POST -H "Content-Type: application/json" -d '{"hanzi_string": "钒氮钠镎铊铥钋"}' https://sevenyan/api/coordinates/
